NX Creative builds FiveM resources for roleplay servers. Weather systems,
job tablets, ATMs, laptops, casino scripts, all designed to drop into an
ESX, QBCore, or QBox server.

This site hosts the documentation: installation guides, configuration
references, exports, commands, events, and per-script changelogs.

## Where to buy

All scripts are sold through our Tebex storefront at
[nxcreative.tech/products](https://nxcreative.tech/products). Licenses
bind to your FiveM server key automatically through Tebex Keymaster, so
there is no manual activation step.
