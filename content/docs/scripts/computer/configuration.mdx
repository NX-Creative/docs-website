description: Every owner-editable entry in config.lua and config_server.lua. Defaults, types, and purpose.
---

All settings live in `config.lua`. The file is in `escrow_ignore`, so your edits
survive updates. It is ordered by how often an owner touches it: Quick Setup first,
then Features, then Advanced. Server-only secrets live in `config_server.lua`, a
server script that never downloads to clients. Keep tokens and API keys there.

### Banking

Backend for the Banking app and all in-app money movement.

<ConfigOption name="Config.Banking.Backend" type="string" default="'auto'" description="Money backend for the Banking app. One of auto, framework, or realbanking.">
- `'auto'` uses nx_realbanking when it is started, otherwise the framework backend. Recommended.
- `'framework'` uses the framework's own cash and bank accounts. Works anywhere with no dependencies.
- `'realbanking'` bridges to nx_realbanking. Proximity-gated, with a clean not-configured state when absent.
</ConfigOption>

<ConfigOption name="Config.Banking.CashAccount" type="string" default="nil" description="Framework-backend cash account name. nil uses stock naming per framework." />

<ConfigOption name="Config.Banking.BankAccount" type="string" default="nil" description="Framework-backend bank account name. nil uses stock naming per framework." />

<ConfigOption name="Config.Banking.MaxTransfer" type="number" default="1000000" description="Largest single transfer, request, or scheduled payment." />

<ConfigOption name="Config.Banking.TransferMemoMaxLen" type="number" default="140" description="Memo character cap. 0 disables memos." />

<ConfigOption name="Config.Banking.MaxTransfersPerMinute" type="number" default="6" description="Per-account outbound burst cap. Guards against a compromised client sweeping an account. The scheduled-payment watcher is exempt. 0 disables it." />

<ConfigOption name="Config.Banking.DailyTransferCap" type="number" default="5000000" description="Per-account outbound total over a trailing 24 hours, in fiat." />

<ConfigOption name="Config.Banking.SchedulePollIntervalMs" type="number" default="60000" description="Framework-backend standing-payment watcher cadence in milliseconds, clamped between 15 seconds and 10 minutes." />

<ConfigOption name="Config.Banking.Adapter" type="table" default="nil" description="Custom money adapter, used only when Config.Framework is set to a custom value. ## Server secrets (config_server.lua)

`config_server.lua` loads only on the server and never ships to clients. It extends the
same `Config` table. Put all secrets and API tokens here.

<ConfigOption name="Config.Media.FiveManageToken" type="string" default="''" description="FiveManage Images API token, the one secret the media and Photos feature needs. Create your own Images token in the FiveManage dashboard under API keys and paste it here. Leaving it empty keeps uploads disabled with a clean not-set-up state.">
Never put this token in `config.lua`, which downloads to every client. It belongs only in `config_server.lua`.
</ConfigOption>
