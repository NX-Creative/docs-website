This section fills up as scripts are published. Each one gets its own
page tree:

- **Overview** — what it does, who it's for, at-a-glance features.
- **Installation** — prerequisites and setup beyond the
  [generic install guide](/docs/getting-started/installation).
- **Configuration** — every entry in `config.lua`.
- **Exports** — client and server exports the script publishes.
- **Commands** — chat commands registered by the script.
- **Events** — net events you can listen for or trigger.
- **Changelog** — version history.

## What's available

Scripts will be listed here as they're documented. For the latest
catalog see the [store](https://nxcreative.tech/products).
