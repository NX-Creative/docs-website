The four games are opened by interacting with a slot cabinet through
`ox_target`. There is no chat command to open a game.

## Admin

<CommandReference
  name="manageslots"
  description="Opens the in-world slot placement UI. Walk to a free spot, pick a cabinet model, and confirm. Placements are persisted to the database. The command name can be renamed via Config.SlotPlacement.Command."
/>

Access is controlled by
[`Config.SlotPlacement.Access`](./configuration#configslotplacementaccessadmins).
By default only `admin` and `superadmin` groups (or matching QBCore /
QBox / Ace permissions) can open the manager. Whitelisted jobs can be
granted access through
[`Config.SlotPlacement.Access.Jobs`](./configuration#configslotplacementaccessjobs).
