title: Configuration
description: Every entry in config.lua. Defaults, types, and purpose.

All settings live in `config.lua`. The file is in `escrow_ignore`, so
it stays editable after install. Server-only Discord webhooks live in
`server/webhooks.lua`, which is covered in
[Installation](./installation#discord-webhooks).

## General

<ConfigOption name="Config.Framework" type="string" default="'esx'" description="Framework integration. One of 'esx', 'qbcore', 'qbox'. 'qbx' is accepted as an alias for 'qbox'." />

<ConfigOption name="Config.Inventory" type="string" default="'ox_inventory'" description="Inventory provider. One of 'ox_inventory', 'qb-inventory'." />

<ConfigOption name="Config.Target" type="string" default="'ox_target'" description="Target provider used for the laptop's open and pickup interactions. One of 'ox_target', 'qb-target'." />

<ConfigOption name="Config.Account" type="string" default="'bank'" description="Which account balances settle trading, casino, marketplace, and darknet transactions against. Typically 'bank', 'money', or 'cash'." />

<ConfigOption name="Config.TableName" type="string" default="'nx_laptop_assets'" description="Database table that stores per-player stock and crypto holdings. Change only if you renamed the imported SQL table." />

<ConfigOption name="Config.LaptopItem" type="string" default="'laptop'" description="Inventory item name that represents the laptop. Must match the item registered in your inventory system." />

## Jobs hub

<ConfigOption name="Config.JobsHub.Enabled" type="boolean" default="true" description="Master toggle for the jobs app. If false, the app is hidden in the UI." />

<ConfigOption name="Config.JobsHub.Jobs" type="table" description="Array of job definitions. Each entry: { id, label, description, requirements, questions }. The id must match a key in ServerConfig.JobWebhooks. Shipped with police, ems, and mechanic. Each questions entry: { id, label, placeholder, required }." />
